- userKey (string)

Backend logic

On:
POST /api/run
POST /api/run/stream
POST /api/build

2. Get today date
3. Fetch or create row
4. If runsCount >= 50 → return 429
5. Else increment runsCount

- limit = 3/day

---

Response format

{

}

{
remaining: X
}
